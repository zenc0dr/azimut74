Краткая инструкция

Пресет виджета можно получить из URL после поискового запроса со стартовой страницы после ?query=

Как можно заметить, в строке пресета присутствуют конкретные даты 01.09.2020-15.09.2020, Вычисляемые даты реализуются с помощью специальных вставок: {"geo_objects":[],"date_of":"{N}","date_to":"{N+10}","days":[],"adults":1,"childrens":[],"list_type":"schedule"}

N - это сегодняшняя дата
+10 - это плюс десять дней

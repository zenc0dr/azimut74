Исключния кают

  1. 1. В данной вкладке указываются названия теплоходов и категорий кают, которые не нужно парсить из сторонних источников
  2. 2. Правописание названия категорий кают и теплоходов, к которым они принадлежат:
    • "А (3)" - Исключить категорию для всех теплоходов
    • "А (3) {11,12,13}" - Разрешить категорию только для теплоходов с этими id
    • "А (3) {!11,12,13}" - Разрешить категорию для всех теплоходов, кроме имеющих данные id
